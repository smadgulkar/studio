# **App Name**: Unitravco Landing

## Core Features:

- Hero Section: A visually appealing hero section with a captivating image or video and a concise headline describing Unitravco's travel experience curation services.
- Services Overview: Display a summary of the different travel experiences Unitravco offers. Include images and brief descriptions for each service.
- Testimonials Section: Display a rotating or scrolling section of customer testimonials with star ratings, using a carousel-style implementation
- Contact Form: Implement a contact form allowing potential customers to reach out to Unitravco. Required fields include name, email, and message.
- Map Display: Embed a map pinpointing Unitravco's location in Madison, NJ.

## Style Guidelines:

- Primary color: A calming blue (#64B5F6), reminiscent of clear skies and serene travel destinations.
- Background color: A light, off-white (#F5F5F5) to provide a clean and modern feel.
- Accent color: A warm orange (#FFAB40) to highlight call-to-action buttons and important information.
- Headline font: 'Playfair', a modern sans-serif, to convey elegance.
- Body font: 'PT Sans', a humanist sans-serif to convey information with personality.
- Use modern, minimalist icons to represent travel destinations and services.
- Implement a clean, responsive grid layout that adapts to different screen sizes.
- Subtle fade-in animations on content load and carousel transitions for a smooth user experience.
